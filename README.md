# 🛠️ Cybersecurity Toolkit

A collection of beginner-to-intermediate cybersecurity and networking tools built using Python.

---

## 📌 Tools Included

### 🔍 Web Info Tool
- Retrieves IP address of a domain
- Fetches HTTP headers
- Useful for basic reconnaissance

### 🌐 Subdomain Scanner
- Finds common subdomains of a target domain
- Uses DNS resolution
- Helps discover hidden services
