# 🌍 IP Geolocation Tracker

A Python tool that retrieves geographical information about an IP address or domain.

---

## 🚀 Features

- Accepts IP or domain input
- Resolves domain to IP
- Fetches geolocation data using API
- Displays location, ISP, and coordinates
